# @vladpazych/dexter-node

Node runtime package for Dexter.

Install this alongside `@vladpazych/dexter` to get the `dexter` binary in a Node repo:

```sh
npm install -D @vladpazych/dexter @vladpazych/dexter-node zod
```

Write commands in `dexter.config.ts` by importing from `@vladpazych/dexter/cli`.

This runtime is currently experimental.
